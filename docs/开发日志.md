# 开发日志

## 2026-07-11 — v0.2 班级费率管理 + 历史记录加载

### 完成事项

- **费率持久化**：`storage.py` 新增 `load_rates()` / `save_rates()`，费率配置保存到 `~/.salary-counter/rates.json`
- **费率管理弹窗**：新建 `ui/rate_dialog.py`，QDialog 内嵌 QTableWidget，支持班级类型的增删改查 + 校验（非空、不重复、时薪 > 0）
- **主窗口集成**：
  - 班级下拉框旁添加"管理"按钮，点击打开费率弹窗
  - 启动时自动加载持久化费率（覆盖 config.py 默认值）
  - 输入新班级时弹窗引导打开费率管理界面，确认后持久化
- **历史记录加载**：新增"加载数据"按钮，选择 JSON 文件加载历史数据，当前有记录时弹窗确认

### 关键决策

- 费率持久化放在 `storage.py` 而非 `config.py`，与 JSON I/O 逻辑统一
- 弹窗内表格直接可编辑（双击进入编辑），不走额外的"编辑模式"切换
- 历史加载只加一个按钮 + 一个槽函数，不复用独立弹窗，保持极简

### 文件变更

| 文件 | 操作 | 说明 |
|---|---|---|
| `codes/storage.py` | 修改 | +load_rates() / +save_rates() |
