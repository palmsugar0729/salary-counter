## 项目背景

开发一款 Windows 桌面应用程序，帮助用户完成每月上课课时统计，并导出为 Excel 文件。当前阶段为个人使用，未来可能扩展为公司同事共用的小程序。

---

## 技术选型讨论

### 最终确定

| 模块 | 选型 | 理由 |
|---|---|---|
| GUI 框架 | PySide6 | MVC/信号槽架构与 uniapp/Vue 的数据驱动模式接近，便于未来移植；QTableView 体验优于 tkinter Treeview；LGPL 协议无商用限制 |
| Excel 生成 | openpyxl | Python 生态最成熟的 Excel 读写库 |
| 数据持久化 | JSON 文件 | 零依赖，实现成本极低；未来可无缝迁移至数据库或云端 API |
| 代码语言 | Python | 与项目定位相符 |

### 关于 tkinter vs PySide6 的决策

- tkinter 写小工具快，但 UI 和数据逻辑容易耦合，迁移 uniapp 时心智转换成本高
- PySide6 的 signal-slot 模式和 Vue 的响应式数据流理念相近，移植时代码结构可直接参考
- PySide6 内置 QTableView，预览表格体验好

### 关于 JSON 持久化的决策

- 不引入数据库（SQLite 也不需要），保持简单
- 当前个人使用场景下，关掉重来也没问题
- 未来需要持久化时数据结构不变，只需换存储层

## Excel 格式规范（基于参考文件解析）

参考文件：`2026沐晨_4月工资结算表-梁筱.xlsx`

### 整体布局（A~K 列）

| 行号 | 内容 |
|---|---|
| 第1行 | 标题"工资结算表"（A1:I1 合并单元格） |
| 第2行 | H2 = "姓名："，I2 = 教师姓名；A2 的日期删除（原文件遗留问题） |
| 第3行 | 表头：日期(A) / 星期(B) / 时间(C) / 小时数(D) / 时薪(元/时)(E,F 合并) / 薪酬(G) / 合计时数(小时)(H) / 合计薪酬(元)(I) / 班级(J) / 备注(K) |
| 第4行起 | 数据行 |
| 末行 | 合计时数(H 列) = SUM(D 列)，合计薪酬(I 列，原文件在 G18) = SUM(G 列) |

### 各列说明

- **日期(A)**：日期格式，如 2026/4/4
- **星期(B)**：根据日期自动计算（一/二/三/四/五/六/日）
- **时间(C)**：开始时间，如 9:30
- **小时数(D)**：浮点数，如 2.5
- **时薪(E+F 合并)**：根据班级人数自动带出
- **薪酬(G)**：Excel 公式 `=D*E`（小时数 × 时薪）
- **合计时数(H)**：仅在最后一行汇总，`=SUM(D列)`
- **合计薪酬(I)**：仅在最后一行汇总，`=SUM(G列)`（原文件放在 G 列末行，修正到此列）
- **班级(J)**：如 1对3、1对4 等
- **备注(K)**：如"补课*2"、"试听"等

### 同班级颜色区分

- "同一班级"的定义：**班级类型 + 星期 + 开始时间** 三个字段完全一致才算同一班级
  - 例如：周六 9:30 1对3 和 周六 18:30 1对3 是两个不同的班级，用不同颜色
- 同班级的数据行在 Excel 中使用相同背景色
- 切换班级时切换颜色，使用一组预定义的颜色循环

## GUI 交互设计

### 输入区域
- 教师姓名：文本输入框
- 上课日期：日期选择器
- 开始时间：时间选择器（或下拉选择）
- 课时数：文本输入框（浮点数）
- 班级类型：下拉菜单（1对2、1对3、1对4、1对5...，可扩展）

### 操作区域
- "添加"按钮：将当前输入添加到预览列表
- "生成 Excel"按钮：导出为 Excel 文件（弹出保存路径对话框）

### 预览区域
- QTableView 显示已添加的所有记录
- 列与 Excel 保持一致
- 自动计算：星期、时薪、薪酬
- 支持删除某条记录（右键或按钮）

## 数据结构设计

### 记录条目（Record）
```python
@dataclass
class Record:
    date: date
    start_time: time
    hours: float          # 课时数
    class_type: str       # 班级类型，如"1对3"
    note: str             # 备注，默认空
```

### 课程配置（可扩展）
```python
CLASS_RATES = {
    "1对1": 90,    # 预留
    "1对2": 100,
    "1对3": 100,
    "1对4": 110,
    "1对5": 120,
}
```

### 月度记录集合
```python
@dataclass
class MonthData:
    teacher_name: str
    records: list[Record]
```
