# 目的
开发一款简单的windows应用程序，用户通过输入、点击等操作，完成本月上课的课时统计，并导出为一个ecxel文件。
# 需求
## 1、图形界面
可交互式设计，界面上需要有如下内容——
- 输入教师姓氏
- 选择上课日期
- 选择开始时间
- 输入课时数（该数字为浮点数）
- 选择课程人数
选择设计成下拉菜单；输入则是输入框。
以及，一个生成excel的button
## 2、生成EXCEL表格
在点击生成excel的button之后，将获取的数据整理成excel表格并导出。

由于现阶段想要做的是个个人用的小程序，因此无需数据库。
